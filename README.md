 # FoodMunch Website

A fully responsive website for a restaurant, designed using HTML, CSS, and Bootstrap.

## Features

- **Responsive Design**: The website adapts to various screen sizes, including mobile, tablet, and desktop.
- **Navigation Bar**: Easy-to-use navigation bar with links to different sections of the site.
- **Menu Section**: Displays the restaurant's menu in an attractive layout.
- **Contact Form**: Provides a way for customers to contact the restaurant.
- **Footer**: Contains links to social media and basic contact information.

## Technologies Used

- **HTML**: Used for the structure of the website.
- **CSS**: Custom styles for the website's look and feel.
- **Bootstrap**: Used for layout and responsive design.
